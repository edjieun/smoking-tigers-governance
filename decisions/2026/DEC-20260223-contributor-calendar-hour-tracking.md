# DEC-20260223-contributor-calendar-hour-tracking

**Decision ID:** DEC-20260223-007
**Title:** Adopt Contributor Calendar Usage and Hour Tracking Policy
**Status:** Approved
**Date:** 2026-02-23
**Owner / Approver:** Ed (Steward)

## Context
Contributors and creators need a lightweight, fair way to use calendar events for work and hour tracking without excessive admin burden.

## Decision
Adopt a contributor calendar policy that defines trackable event categories, minimum event metadata, and creator-friendly tracking standards.

## Rationale
- Provides operational visibility into work commitments
- Supports budgeting and staffing decisions
- Respects creator workflows while preserving accountability

## Tradeoffs / Risks
- Risk of over-tracking if not managed carefully
- Calendar data alone may not reflect actual work quality

## Impacts
- Contributors performing scheduled work must use trackable calendar events when hour tracking is required
- Project leads define required tracking categories per project

## Dependencies
- Shared calendars policy
- Compensation and RevPoint policies (separate)
