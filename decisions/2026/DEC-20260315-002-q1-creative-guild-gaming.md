## Context

Zeb, a video game designer/developer previously interested in building a guild within Q1, is being engaged to launch the first gaming chapter of the Q1 Creative Guild. Basil (Creative Director) is responsible for fostering the creative community layer across all guilds.

## Decision

**Authorize the launch of a Q1 Creative Guild — Gaming Chapter, with Zeb as founding steward candidate.** The guild will operate under Q1 governance principles with a private GitHub repo, RP-based contributor economics, and a mission focused on building original gaming IP.

## Structure

- **Model:** Q1 Creative Guild (governed contributor community)
- **Focus:** Game development, tools, content, IP creation
- **Governance:** Private GitHub repo forked from Q1 governance template
- **Economics:** Revenue Points — purchased at face value, earned through contributions, redeemable against project revenue
- **Creative Director liaison:** Basil

## Launch Playbook (4 Steps)

1. **Form the core:** 3–4 founding members with shared purpose; define mission
2. **Launch governance repo:** Private GitHub repo tracking decisions, members, projects
3. **Start building IP:** First projects scoped and owned; contributions logged
4. **Activate RP:** Members purchase and earn RP; dual-currency model goes live

## Rationale

- Gaming guild is a natural entry point for the Q1 creative community strategy
- Zeb brings domain expertise and existing relationships in the game dev space
- The guild model validates the RP + governance framework in a new vertical
- Discord is the obvious community surface for a gaming guild — connects cleanly to the multi-channel comms strategy
