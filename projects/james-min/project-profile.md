## The Bottleneck

Editing is the primary constraint. Each video requires:
- Hundreds of cuts
- YouTube copyright compliance (evasion of audio/video detection)
- Three layout layers: (1) full screen show + face circle overlay, (2) full face + show inset, (3) blurred background + face centered
- Thumbnail creation and A/B testing
- Manual re-upload cycles when YouTube flags content

This creates a massive backlog: Gilded Age S1, Crown S4, Stranger Things, ~20 movies — none yet on YouTube.

## Production Workflow

**Capture:**
- Two MP4 files: face cam (DSLR, audio included) + stream capture (1080p, Netflix/Amazon in browser)
- Recorded via OBS; separate scenes
- File sizes: 5–9 GB (face cam), 3–7 GB (stream) per movie

**Edit:**
- Tool: CapCut (switched from Premiere)
- Storage: Dropbox
- Branded assets: bumpers, logo, meme clips stored in CapCut branded media cloud

**Distribution:**
- Upload to Patreon (raw, fast)
- Edit → upload to YouTube as private → wait for copyright scan → make public if cleared

---

## Content Agent Pipeline Scope

Specialized agents to handle:
1. **Ingest** — receive raw MP4 files from James's shared Drive
2. **Edit** — cut, layer, comply (copyright-safe slicing, zoom/mirror techniques)
3. **Compliance** — track copyright exposure per video (target: ≤10 min total, ≤5s clips)
4. **Copy** — generate titles, descriptions, thumbnails
5. **Publish** — upload to YouTube (private), monitor copyright scan, make public
6. **Repurpose** — clips for Patreon, social
