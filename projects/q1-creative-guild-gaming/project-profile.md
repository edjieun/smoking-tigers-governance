## Launch Playbook

### Step 1 — Form the Core
Recruit 3–4 founding members with shared creative purpose. Define the mission and what counts as a contribution. The founding group sets cultural and governance DNA.

### Step 2 — Launch a Private Governance Repo
Fork Q1 governance model into a private GitHub repo:
```
/decisions/       — DACI-style decision log
/members/         — directory + roles
/projects/        — active project profiles
/governance/      — rules, policies, contribution standards
README.md         — mission and membership criteria
```

### Step 3 — Start Building IP
First projects scoped and owned. Contributions logged. RP earned per contribution. Project types: game concepts, full builds, tools, content, community infrastructure.

### Step 4 — Revenue Points Activation
Members purchase RP at face value (staking into the mission). RP earned through verified contributions. Redeemable against project revenue. Dual-currency model: cash for ops, RP for contribution alignment.
