# smoking-tigers-governance

Governance repository for Smoking Tigers Enterprises (STE).

This repo contains the canonical governance documents, policies, charters, project documentation, and the Knowledge Bible that define how STE operates.

## Structure

```
├── docs/              Governance overview, glossary, enterprise docs
├── charters/          Organizational charter
├── policies/          Document status, role authority, RevPoint policy
├── agreements/        Contribution and partnership agreements (pending)
├── projects/          Per-project documentation
│   └── smoking-tigers-ai/
├── knowledge-bible/   STE Knowledge Bible (canonical reference)
└── _incoming/         Quarantine for unvalidated files
```
